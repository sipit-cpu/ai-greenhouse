# Smart Mini Greenhouse (IoT + AI)

A mini greenhouse system with IoT monitoring and basic AI automation.

## Sensors
- Capacitive Soil Moisture Sensor
- DHT22 Temperature & Humidity Sensor
- BH1750 Light Intensity Sensor

## Actuators
- 5V DC Water Pump
- 5V UV Grow Light
- 5V DC Fan

## Hardware
- ESP8266
- ESP8266 Expansion Board
- 4-Channel Relay Module
